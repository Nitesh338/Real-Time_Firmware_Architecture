This project is for educational purpose

This is the bare-metal FreeRTOS integrated Plateform SOfrware 
Key features developed in this project are:
    > You can create any number of blocks in appl_main.c file and add  config in blockconfig table
    > Reset handling is implemented , if any reset happens , hardfault reason is stored in the variables from the register.
    > Tasks are created , one 10ms tas is created and NvM is running in that task.
